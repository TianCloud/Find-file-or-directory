sample of Running the program 'myFind.c':

---------------Compiling--------------
	gcc myFind.c
		or,
	gcc -o myFind myFind.c

---------------Executing---------------
if not using -o command:
	./a.out filename.txt
	./a.out directoryName

	./a.out filename.c directoryName
	./a.out directoryName otherDirectoryName

if using -o myFind:
	./myFind filename.txt
	./myFind directoryName

	./myFind filename.c directoryName
	./myFind directoryName otherDIrectoryName
